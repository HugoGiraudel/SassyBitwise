    bitwise 

    @function bitwise($args) { ... }
    View source

    Description

    Bitwise helper function

    Parameters

    parameter Name parameter Description parameter Type parameter Default value
    $args

    list of parameters

    		 List — none

    Returns

    Number, False

    Example

    AND

    $value: bitwise(42 '&' 48);
// -> 32

    OR

    $value: bitwise(42 '|' 48);
// -> 58

    XOR

    $value: bitwise(42 '^' 48);
// -> 26

    NOT

    $value: bitwise('~' 42);
// -> -43

    SHIFTLEFT

    $value: bitwise(42 '<<' 48);
// -> 168

    SHIFTRIGHT

    $value: bitwise(42 '>>' 48);
// -> 10
